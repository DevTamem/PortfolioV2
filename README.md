# Portfolio Website

My personal portfolio website showcasing my projects, skills, and experience as a Software Engineering student.

**Live Demo**: [https://portfolio-v2-lac.vercel.app/](https://portfolio-v2-lac.vercel.app/)

## About

This is my portfolio website where I share my journey as a developer. It features my projects, technical skills, and provides a way to get in touch with me.

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React

## Development

To run this project locally:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

## Build

```bash
npm run build
```
